= 删除内部 ONTAP 工作环境
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
如果您不想再从 Cloud Manager 管理内部 ONTAP 工作环境，请将其删除。

删除工作环境不会影响 ONTAP 集群。您可以随时从 Cloud Manager 重新发现它。

.步骤
. 在 " 画布 " 页面上，双击内部 ONTAP 工作环境的名称。
. 单击菜单图标并选择 * 删除 * 。
+
image:screenshot_remove_onprem.png["显示页面右上角菜单和删除选项的屏幕截图。"]

. 单击 * 删除 * 进行确认。
